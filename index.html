¿Por donde empezar?

Un día común como todos los demás estaba navegando por las redes sociales y me encuentro con una publicación de mi facultad acerca de un programa de Microsoft llamado Innavacción Virtual. Entre por curiosidad y hasta el día de hoy ha sido una de las mejores experiencias que he tenido en mi vida.

Conceptos que he aprendido durante mi camino en el programa

Cloud

Imagínalo como un ecosistema de servidores que te da las herramientas que necesites realizar cualquier tipo de tarea que desees hacer. Existen múltiples servicios de nube como Azure, AWS y GCloud

GitHub

Es una Red social que utiliza los principios del sistema de control de versiones Git para que puedas colaborar con personas de todo el mundo en múltiples proyectos

Bot Services

Son plantillas que nos proporciona Azure con las cuales podemos construir, implementar y administrar bots

Artificial Intelligence

Es una rama científica que se encarga de crear mecanismos y programas que puedan mostrar comportamientos inteligentes.

Virtual Machine

Son aplicaciones que virtualizan hardware y software. Las usamos cuando necesitamos el control completo sobre un entorno.

Cloud Storage

Imaginalo como un ecosistema de servidores que te da las herramientas para que desees realizar cualquier tipo de tarea que necesites. 

Azure Search

Utiliza A.I para extraer información valiosa de muchos datos. Como patrones, sentimientos, relaciones. etc.

Serverless Computing

Básicamente no ten tienes que encargar de administrar el servidor o la infraestructura que utilizaras, ya que de eso se encarga Azure.

Azure Cognitive Services

Es una familia de servicios de A.I y API cognitivas que nos permiten crear aplicaciones inteligentes.

Vision

 Es un servicio cognitivo que nos permite analizar y identificar con presición el contenido de las imágenes y los vídeos. 

Speak

Es un servicio cognitivo que mediante el uso de AI te permite convertir voz a texto, producir una voz con sonido natural en base a un texto

Language

Es un  servicio cognitivo que nos permiten comprender el significado del  un texto no estructurado o reconocer la intención del hablante.



Knowledge

Son herramientas del SDK que nos permiten crear recursos de conocimiento para integrarlos a aplicaciones o servicios

Containers

Son aplicaciones que virtualizan un sistema operativo, a diferencia de las V.M estas no simulan hardware y pueden llegar a ser mejores que las V.M en algunos casos

Disponibility

Es el tiempo que esta disponible un servicio sin ser interrumpido.

Resistance

Es la capacidad de un sistema de permanecer disponible en ocasiones irregulares como: 
-Desastres naturales
-Altos niveles de trafico
-Ataques al sistema
